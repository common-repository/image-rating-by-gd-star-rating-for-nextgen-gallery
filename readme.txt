== Description ==

**The plugin renders the GD Star Rating shortcode in a way suitable for the Nextgen Gallery.**     
*The "Image Rating by GD Star Rating for Nextgen Gallery" plugin is useful if you are building a public gallery       
where people can bring their own contribution.*  


== Installation ==

1. Extract the zip file.
1. Upload the folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
